# 任務完成報告：門診時刻表 (Schedule) 頁面製作

## 1. 任務概要
依據您的指示，已全新製作了 **[http://localhost:3000/schedule](http://localhost:3000/schedule)** 頁面。
本頁面參照了您提供的參考網站架構，並完全沿用本專案的 **Emerald (翡翠綠)** 視覺風格，確保與首頁及其他頁面的一致性。

## 2. 頁面特色與實作細節

### A. 視覺風格 (Visual Design)
-   **Hero Header**：使用與首頁一致的深綠色漸層背景 (`bg-emerald-900`)，搭配微隱的方塊紋理，傳遞專業與沉穩的醫院形象。
-   **配色系統**：
    -   **主色調**：Emerald-600 用於按鈕與強調色。
    -   **輔助色**：Slate-50 作為背景底色，讓閱讀更舒適。
    -   **狀態色**：使用 Amber-50/400 製作醒目的「注意事項」區塊，提醒掛號規則。

### B. 功能實作 (Frontend Only)
由於不改動後端，我使用了前端模擬資料 (Mock Data) 來構建這個頁面，但架構上預留了未來接科後端的彈性。

1.  **科別篩選 (Department Tabs)**：
    -   製作了「全部科別」、「內科」、「外科」、「兒科」、「婦產科」等切換標籤。
    -   **互動體驗**：使用 `Framer Motion` 實作切換時的按鈕縮放與顏色過渡動畫，點擊不同科別時，下方表格會即時過濾顯示對應醫師。

2.  **每週課表 (Weekly Schedule Grid)**：
    -   **RWD 響應式表格**：設計了可水平捲動的表格，確保在手機上也能完整檢視。
    -   **時段劃分**：分為早診、午診、晚診，並清楚標示時間。
    -   **醫師卡片**：
        -   依據科別自動變色（例如：內科為藍色系、外科為琥珀色系、婦產科為綠色系），方便視覺辨識。
        -   滑鼠懸停 (Hover) 時有上浮效果，提升質感。

3.  **實用資訊**：
    -   加入了 PDF 下載按鈕（UI 示意）。
    -   加入了掛號注意事項提示框。
